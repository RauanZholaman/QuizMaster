# QuizMaster - Full Stack Quiz Application

This directory is a Monorepo* for project QuizMaster, containing both backend and frontend applications.

## Project Structure
- `quizmaster-backend/` - Spring Boot backend application
- `quizmaster-frontend/` - React frontend application with AI-powered question generation

## 🚀 Quick Start for Teammates

### Frontend Setup (5 minutes)
```bash
cd quizmaster-frontend
npm install
cp .env.example .env
npm start
```

📖 **For detailed setup instructions**: See [SETUP_GUIDE.md](SETUP_GUIDE.md)  
⚡ **For quick reference**: See [QUICK_START.md](QUICK_START.md)

## Features
- ✅ **Auto Question Generation**: Dynamic count-based question generation
- ✅ **Multiple Question Types**: MCQ, True/False, Short Answer, Fill Blank
- ✅ **Realistic Sample Questions**: 40+ educational questions for testing
- ⏸️ **AI Integration**: Ready for Gemini AI (currently using samples)
- ✅ **Firebase Integration**: Backend data storage
- ✅ **Modern UI**: React with responsive design

## Current Status
